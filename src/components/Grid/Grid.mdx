# Grid
- Mui의 Grid 컴포넌트를 사용합니다.
- Grid는 Container, Item로 구분됩니다. Container 내 Item을 넣어서 사용합니다.
- Grid Prop 중 Container: true 값을 적용하면, 디바이스 별 Container 스타일이 적용됩니다.

---

### Desktop (768px ~)
- 최대 너비 1380px
- 양옆 Padding 40px
- Gap 24px

### Mobile (0px ~ 767px)
- 양옆 Padding 16px
- Gap 24px
