# Button

ACON3D 서비스에 사용되는 버튼입니다. 기본적으로 정해진 preset을 사용하되, 필요에 따라 `sx` prop을 통해 스타일을 커스텀할 수 있습니다.

옵션 선택이 유효하지 않을 시, 개발자 도구의 콘솔 창에 경고 메시지가 출력됩니다.
