# BottomSheet

BottomSheet 컴포넌트는 하단에서 슬라이드되어 컨텐츠를 표시하며, 주로 모바일에서 사용됩니다.

### Props

- `height`: 컴포넌트 세로 길이를 결정합니다. `full`은 현재 화면의 90%까지 채우며, `hug`는 콘텐츠의 길이에 맞게 설정합니다.
- `showCloseIcon`: BottomSheet 컴포넌트 상단 우측에 닫기 아이콘을 표시할지 여부를 결정합니다. (default `true`)
- `renderTitle`: 제목을 표시하는 영역을 렌더링하는 함수입니다. **빈 값일경우 제목 영역이 표시되지 않습니다.**
- `renderContent`: 콘텐츠를 표시하는 영역을 렌더링하는 함수입니다. **빈 값일경우 콘텐츠 영역이 표시되지 않습니다.**
- `renderActions`: 액션 버튼을 표시하는 영역을 렌더링하는 함수입니다. **빈 값일경우 액션 버튼 영역이 표시되지 않습니다.**

### CSS Classes

BottomSheet 컴포넌트는 아래와 같은 Class를 제공하며, 해당 클래스들을 사용하여 커스텀 스타일을 적용할 수 있습니다.
- `.bottomSheet-header`: 닫기 버튼을 포함하고 있는 헤더 영역입니다. `showCloseIcon`이 false일 경우에는 헤더 영역이 표시되지 않습니다.
- `.bottomSheet-closeIcon`: 닫기 아이콘을 감싸고 있는 영역입니다. `showCloseIcon`이 false일 경우에는 헤더 영역이 표시되지 않습니다.
- `.bottomSheet-title`: 제목을 표시하는 영역입니다.
- `.bottomSheet-content`: 컨텐츠를 표시하는 영역입니다.
- `.bottomSheet-actions`: 액션 버튼을 표시하는 영역입니다.
