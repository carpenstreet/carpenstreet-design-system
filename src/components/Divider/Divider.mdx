# Divider

눈에 잘 띄지 않는 얇은 선으로 컨텐츠를 구분할 때 사용합니다.

기본적으로 너비 또는 높이는 100%로 설정되어 있습니다.

### Props
- `light` true일 경우 색상이 더 밝아집니다. (필수)
- `orientation` 옵션은 `horizontal`과 `vertical`이 있습니다. (필수)
- `sx` 옵션으로 Divider 스타일을 커스텀할 수 있습니다.
