# Avatar

유저의 프로필을 표시하는 UI입니다.

`src` prop에 image url을 입력하여 프로필 이미지를 표시할 수 있으며, `src`를 입력하지 않으면 `ProfileFilled` 아이콘이 표시됩니다.

`src`에 입력된 image url을 통해 이미지를 받아오는데 실패하면 `alt`에 입력한 텍스트가 안내됩니다. (HTML의 `<img>`태그 참고)

MUI `<Avatar>` 컴포넌트의 다른 prop 또한 사용할 수 있습니다.
