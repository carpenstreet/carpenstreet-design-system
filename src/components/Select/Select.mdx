# Select

선택한 값을 입력받는 UI입니다.

`<Select>`는 **카펜스트리트 디자인시스템**의 `<TextField>` 컴포넌트를 사용합니다.

`<TextField>` 컴포넌트와 마찬가지로 MUI의 `<TextField>` API 대부분을 그대로 사용할 수 있습니다. 단, 아래 props는 사용할 수 없습니다.

- `variant` : `outlined`로 고정되어 있습니다.
- `select` : `true`로 고정되어 있습니다.
- `color` : 사용하지 않는 prop입니다.

MUI의 `<TextField>` API 이외에 `success`, `withHelperTextIcon` props가 추가되었습니다.
