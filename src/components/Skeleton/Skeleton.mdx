# Skeleton

MUI의 Box 컴포넌트를 이용해 직접 스타일링하여 제작된 컴포넌트입니다.

MUI의 기본 prop들을 사용할 수 있습니다.

<Canvas of={SkeletonStories.Default} />

<Controls include={['variant', 'width', 'height']} />

### Text Skeleton

<Canvas of={SkeletonStories.TextSkeleton} />

텍스트의 경우 세로 너비를 지정하는 방법이 아래 2가지인데, 필요한 상황에 맞춰서 사용 가능합니다.
- Skeleton의 height prop을 이용해 직접 지정
- Text Skeleton의 경우 typography의 variant를 넣어서 텍스트의 크기에 맞춰 자동으로 세로 길이가 조절

```jsx
  <Skeleton variant="text" sx={{ typography: 'typography/headline/h1/black' }} />
```

### Circular Skeleton

Circular는 width 값만 적용할 수 있으며, width를 설정하면 height 스타일도 동일하게 적용됩니다.
