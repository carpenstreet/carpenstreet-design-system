**카펜스트리트 디자인 시스템** 스토리북에 오신 것을 환영합니다!

스토리북에서는 디자인 시스템에 어떤 컴포넌트가 존재하고, 어떻게 동작하는지 직접 제어하며 확인해볼 수 있습니다.

현재 카펜스트리트의 디자인 시스템은 Product Designer와 Frontend Engineer가 서로 긴밀하게 협업하며 구현 중입니다!

디자인 시스템은 MUI를 기반으로 Customize하여 디자인/구현되었으니, 기능 및 디자인에 어려움을 겪으신다면 MUI를 참고해주세요!

## Getting Started - for Devs

### Installation

우선, git repo를 clone해주세요

```bash
# install dependency
yarn install
```

### Running the app

```bash
# run development mode
yarn dev
```
