**카펜스트리트 디자인 시스템** 스토리북에 오신 것을 환영합니다!

스토리북에서는 디자인 시스템에 어떤 컴포넌트가 존재하고, 어떻게 동작하는지 직접 제어하며 확인해볼 수 있습니다.

현재 카펜스트리트의 디자인 시스템은 Product Designer와 Frontend Engineer가 서로 긴밀하게 협업하며 구현 중입니다!

디자인 시스템은 MUI를 기반으로 Customize하여 디자인/구현되었으니, 기능 및 디자인에 어려움을 겪으신다면 MUI를 참고해주세요!

## Getting Started - for Devs

### 스토리북 레포 설치 및 실행

우선, git repo를 clone해주세요

```bash
# install dependency
yarn install
```

그리고 아래 명령어로 스토리북을 실행해주세요

```bash
# run development mode
yarn dev
```

### 프로젝트에 디자인 시스템 패키지 설치 및 적용

```bash
# install package
yarn add carpenstreet-designsystem
```

theme.d.ts 파일을 생성하여 아래와 같이 작성해주세요

```typescript
/// <reference path="[node_modules 상대 경로]/carpenstreet-designsystem/dist/src/shared/settings/theme.d.ts" />
```

DesignSystem을 사용하고자 하는 컴포넌트 상위에 Provider를 추가해주세요

```tsx
import { DesignSystemProvider } from 'carpenstreet-designsystem';

...
  <DesignSystemProvider>
    <Something />
  </DesignSystemProvider>
...
```

theme 객체를 사용하실 땐 `useDesignSystemTheme()` hook을 사용해주세요

```tsx
import { useDesignSystemTheme } from 'carpenstreet-designsystem';

...
  const theme = useDesignSystemTheme();
  const color = theme.palette['color/primary/600'];
...
```
