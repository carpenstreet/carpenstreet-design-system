# Responsive

컴포넌트의 반응형 처리에 대한 가이드라인입니다.

반응형 처리는 아래와 같은 방법으로 통일하고 있으며, 이를 준수하여 개발해주시기 바랍니다.

---

### Typography
Typography는 지정된 breakpoint에 따라 variant를 지정할 수 있습니다.

[Typography 스토리북 페이지](/docs/foundation-typography--docs#responsive)에서 더 자세하게 확인이 가능합니다.
```tsx
<Typography
  sx={{
    typography: {
      desktop: 'typography/headline/h1/black',
      mobile: 'typography/title/small/black'
    }
  }}
  variant="typography/headline/h1/black"
>
  안녕하세요
</Typography>
```

### Typography를 제외한 나머지 컴포넌트
기본으로 노출할 breakpoint(ex: desktop)만 컴포넌트의 prop을 지정합니다.

이외의 breakpoint(ex: tablet | mobile ...)들은 sx prop을 통해 직접 Style을 지정합니다.

다음은 예시 코드입니다.
```tsx
<Button
  variant={'text'}
  size={'M'}
  sx={{
    '@media (max-width: 1200)': {
      fontSize: '12px',
    },
    '@media (max-width: 600)': {
      fontSize: '10px',
    }
  }}
/>
```
